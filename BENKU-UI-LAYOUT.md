# BENKU App UI 布局规范文档

## 一、整体架构

- **页面数量**：4 个全屏页面
  1. **ChatScreen**（首页/对话页）
  2. **FavoritesScreen**（收藏夹页面）
  3. **PlayerScreen**（磨耳朵播放页面）
  4. **SettingsScreen**（设置页面）

- **导航方式**：Modal 全屏覆盖，无底部 TabBar
  - 从 ChatScreen → 左上角返回 → FavoritesScreen
  - 从 FavoritesScreen → 右上角设置按钮 → SettingsScreen
  - 从 FavoritesScreen → 底部悬浮按钮 → PlayerScreen
  - 所有弹窗均为半透明遮罩 + 居中卡片

---

## 二、页面详细布局

### 1. ChatScreen（首页/对话页）

#### 顶栏（Header）
- **高度**：56px
- **背景**：半透明白色 `rgba(255,255,255,0.7)` + 模糊效果
- **底部边框**：1px 极细 `#E5E7EB`

**布局（flexbox，justifyContent: space-between）**：

| 位置 | 元素 | 样式 |
|------|------|------|
| 左侧 | 返回图标按钮 | 40x40px 点击区域，图标 20px 颜色 `#FFD43D` |
| 中间 | 标题 + 模型选择器 | 水平排列，gap: 6px |
| 中间-标题 | "BENKU" 文字 | 字号 16px，字重 600，颜色 `#111827` |
| 中间-模型 | 下拉选择器（图标 25px + ▼ 箭头 7px） | 点击弹出模型菜单 |
| 右侧 | 菜单图标按钮 | 40x40px 点击区域，图标 20px 颜色 `#FFD43D` |

#### 对话区域（FlatList）
- **位置**：顶栏下方，固定在屏幕中央，铺满剩余空间
- **内边距**：16px 水平
- **滚动**：自动滚动到底部（新消息）

**气泡样式**：

| 类型 | 背景色 | 文字色 | 左边框 |
|------|--------|--------|--------|
| 用户发送 | `#FFF9E6` | `#1F2937` | 无 |
| AI 回复 | `#FFF9E6` | `#111827` | 2px `#FFD43D` |

#### 快捷操作按钮栏（QuickRow）
- **位置**：输入框上方
- **布局**：水平排列，gap: 12px
- **按钮样式**：
  - 未选中：背景 `#F3F4F6`，文字 `#6B7280`
  - 选中：背景 `#FFD43D`，文字 `#1F2937`
  - 内边距：10px 垂直，16px 水平
  - 圆角：20px

**按钮列表**：
1. "日语怎么说"（带 ▼ 下拉箭头，点击可切换语言）
2. "什么意思"
3. "自由提问"

#### 输入框栏（InputRow）
- **位置**：屏幕底部（安全区上方）
- **布局**：水平排列

| 元素 | 样式 |
|------|------|
| 输入框 | 背景白色，圆角 24px，高度 48px，左侧 16px 内边距，占满剩余空间 |
| 占位符 | 模式=free 时显示 "来聊聊吧"，否则显示 "输入想问的句子或单词" |
| 发送按钮 | 圆形 44x44px，背景 `#FFD43D`，图标为纸飞机 22px 颜色 `#1F2937` |

#### 语言快速切换菜单（QuickLangMenu）
- **触发**：点击 "日语怎么说" 按钮（已选中状态时）
- **样式**：半透明遮罩 + 白色圆角卡片
- **菜单位置**：按钮下方
- **选项**：
  - 日语、英语、韩语、法语、德语、西班牙语
  - 格式："X 语怎么说"
  - 选中项背景 `#FFD43D`

### 2. FavoritesScreen（收藏夹页面）

#### 顶栏（Header）
- **高度**：56px
- **背景**：同 ChatScreen

**布局**：

| 位置 | 元素 |
|------|------|
| 左侧 | 设置图标按钮（20px，颜色 `#FFD43D`） |
| 中间 | "收藏夹" 标题（16px，600） |
| 右侧 | 排序按钮（↓ 或 ↑）+ "创建收藏夹" 文字按钮 |

#### 收藏夹标签栏（FolderRow）
- **位置**：顶栏下方
- **布局**：水平排列，gap: 8px，可换行
- **内边距**：16px 水平，10px 垂直
- **标签样式**：
  - 未选中：背景 `#F3F4F6`，文字 `#6B7280`
  - 选中：背景 `#FFD43D`，文字 `#1F2937`
  - 圆角：16px
  - 内边距：6px 垂直，14px 水平
- **默认选项**："全部"

#### 句子列表（FlatList）
- **位置**：标签栏下方，铺满剩余空间
- **每项样式**：
  - 背景白色卡片，圆角 12px
  - 内边距 14px
  - 下方 8px 间距
  - 左侧显示 ⭐ 星星（已掌握）
  - 右侧显示日期

**卡片内容**：
- 上方：句子原文（大字 16px 600）
- 中间：假名注音（小字 12px）
- 下方：翻译（小字 14px）

#### 磨耳朵悬浮按钮（PlayerFab）
- **位置**：底部居中，安全区上方 24px
- **样式**：圆角 28px，背景 `#FFD43D`
- **内边距**：14px 垂直，28px 水平
- **内容**：耳机图标（18px）+ "磨耳朵" 文字（14px 700，颜色 `#1F2937`）
- **阴影**：黑色 15% 透明度，偏移 (0, 4)，模糊 12px

#### 创建/编辑收藏夹弹窗
- **遮罩**：半透明黑色 `rgba(0,0,0,0.4)`
- **卡片**：白色，圆角 16px，内边距 20px，居中
- **标题**：16px 700
- **输入框**：圆角 12px，边框 `#E5E7EB`，背景 `#F3F4F6`
- **按钮行**：
  - "取消"：背景 `#F3F4F6`，文字 `#6B7280`
  - "创建"/"保存"：背景 `#FFD43D`，文字 `#1F2937`

#### 长按菜单（MenuModal）
- **选项**：
  - "移动到收藏夹"
  - "标记已掌握" / "标记未掌握"
  - "删除"

### 3. PlayerScreen（磨耳朵页面）

#### 顶栏（Header）
- **高度**：自适应（内边距 12px 垂直）
- **布局**：space-between

| 位置 | 元素 |
|------|------|
| 左侧 | 关闭按钮 ✕（20px，颜色 `#6B7280`） |
| 中间 | "磨耳朵" 标题（18px 700） |
| 右侧 | 播放进度 "1/10"（14px，颜色 `#6B7280`） |

#### 当前播放区域（NowPlaying）
- **位置**：顶栏下方
- **背景**：白色
- **底部边框**：1px
- **内容（居中对齐）**：
  - 句子原文：22px 700，`#111827`
  - 假名注音：16px，`#6B7280`
  - 翻译：15px，`#6B7280`
  - 播放遍数/语速：13px，`#9CA3AF`

#### 句子列表（FlatList）
- **位置**：NowPlaying 下方，铺满剩余空间（底部留 120px 给控制栏）
- **每项样式**：
  - 圆角 12px，白色背景
  - 选中项：背景 `#FFF8E1`，边框 1px `#FFD43D`
  - 左侧：原文（15px 600）
  - 右侧：翻译（13px）

#### 播放控制栏（Controls）
- **位置**：屏幕底部，绝对定位，距底部 40px
- **布局**：水平居中，gap: 24px

| 按钮 | 样式 |
|------|------|
| 上一首 | 56x56px，圆角 28px，背景 `#F3F4F6`，图标 ⏮ 22px |
| 播放/暂停 | 72x72px，圆角 36px，背景 `#FFD43D`，图标 ▶️/⏸ 28px |
| 下一首 | 56x56px，圆角 28px，背景 `#F3F4F6`，图标 ⏭ 22px |

### 4. SettingsScreen（设置页面）

#### 顶栏（Header）
- **同 PlayerScreen 顶栏**
- 标题："设置"

#### 设置内容（ScrollView）
- **内边距**：16px
- **分段标题**：13px 700，颜色 `#6B7280`，大写字母，顶部 20px 边距

**设置卡片**：
- 背景：白色
- 圆角：16px
- 边框：1px `#E5E7EB`
- 内边距：16px
- 行高：44px（含上下 10px padding）

**各分区内容**：

| 分区 | 选项 | 控件类型 |
|------|------|----------|
| 账号 | 邮箱 | 纯文本 "demo@benku.app" |
| 语音 | 播放翻译 | 开关 Switch |
| 语音 | 同一句间隔 (秒) | 步进器 [- 3 +] |
| 语音 | 不同句间隔 (秒) | 步进器 [- 3 +] |
| 语音 | 原文播放次数 | 步进器 [- 4 +] |
| 语音 | 语速序列 | 文本输入框 |
| 语音 | 第二遍降低音量 | 开关 Switch |
| 通用 | 母语 | 单选芯片 [中文繁体] [日文] |
| 通用 | 学习语言 | 单选芯片 [日语] [英语] [韩语]... |
| 通用 | 性别 | 单选芯片 [男性] [女性] |
| 通用 | 沉浸模式 | 开关 Switch |
| 数据 | 累计播放时长 | 纯文本 "X 小时 Y 分钟" |
| 数据 | 清除所有缓存 | 红色按钮 |
| 关于 | BENKU v1.0.0 | 标题 + 介绍文字 |
| 理念 | 理念说明文字 | 多行文本 |
| 底部链接 | 开源许可 / 隐私政策 / 用户协议 | 链接文字 |

#### 芯片样式（Chip）
- 未选中：背景 `#F3F4F6`，文字 `#6B7280`
- 选中：背景 `#FFD43D`，文字 `#1F2937`
- 圆角：16px
- 内边距：6px 垂直，12px 水平

#### 步进器样式（Stepper）
- 按钮：32x32px，圆角 16px，背景 `#F3F4F6`
- 值：15px 600，最小宽度 24px 居中
